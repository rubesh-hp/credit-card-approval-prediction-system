# Credit Card Approval Prediction System 💳🤖

This project is a Machine Learning–based web application that predicts whether a credit card application will be approved or rejected based on applicant details.

It uses a Random Forest Classifier for prediction and a Streamlit interface for a simple, interactive user experience.  
This project was built as a first step in my Data Science journey.

---

## 🚀 Features

- End-to-end Machine Learning pipeline
- Data preprocessing and feature encoding
- Random Forest classification model
- Simple and interactive Streamlit UI
- Real-time prediction based on user input

---

## 🧠 Tech Stack

- Python
- Pandas
- Scikit-learn
- Joblib
- Streamlit

---

## 📊 Model Overview

- Algorithm: Random Forest Classifier
- Problem Type: Binary Classification
- Target: Credit Card Approval (Approved / Not Approved)
- Evaluation Metric: Accuracy
